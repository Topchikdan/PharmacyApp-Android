# PharmacyApp Android

**Мобильное приложение для управления аптекой на платформе Android.**

Удобный инструмент на **Kotlin** для работы с каталогом лекарств, управления запасами и обработки заказов прямо со смартфона.

## ✨ Основные возможности

- Каталог лекарственных препаратов
- Поиск и фильтрация товаров
- Управление остатками и запасами
- Просмотр информации о препаратах
- Удобный и интуитивный интерфейс
- Быстрая навигация между разделами

## 🛠 Технологии

- **Kotlin** — основной язык разработки
- **Android SDK**
- **Gradle (Kotlin DSL)** — система сборки
- Современная архитектура Android-приложений

## 🚀 Как запустить проект

# Клонировать репозиторий
git clone https://github.com/Topchikdan/PharmacyApp-Android.git

# Перейти в папку проекта
cd PharmacyApp-Android

Открыть проект в **Android Studio**, дождаться синхронизации Gradle и запустить на эмуляторе или физическом устройстве.

## 📋 Требования

- Android Studio Hedgehog или новее
- Android SDK 24+
- JDK 11+
